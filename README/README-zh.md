# WebDown 项目说明（中文）

## 项目作用

WebDown 是一个面向网页离线归档的浏览器扩展与授权服务组合项目，核心目标是：
- 将网页内容结构化抓取并打包为可分发、可离线打开的 ZIP 文件
- 提供页面级和站点级两种归档模式
- 通过授权/试用机制管理插件可用性
- 通过更新公告机制向用户推送版本与更新信息

## 功能模块

### 1) 页面归档
用于保存当前页面及其依赖资源，生成离线可访问包。

功能点：
- 抓取当前页面 HTML
- 解析并下载页面资源（CSS、JS、图片、字体、媒体）
- 重写资源引用路径，确保离线可用
- 输出 ZIP 文件

### 2) 整站抓取
用于按链接深度递归抓取站点页面并统一打包。

功能点：
- 从起始 URL 开始广度遍历
- 按抓取深度控制页面扩展范围
- 按页面范围限制域名边界
- 页面与资源统一映射、去重、打包

### 3) 抓取任务控制
用于控制长任务执行过程。

功能点：
- 开始抓取
- 暂停抓取（保留当前已下载内容）
- 继续抓取（从暂停进度恢复）
- 状态与统计实时更新

### 4) 授权与试用
用于控制插件使用权限。

功能点：
- 激活码激活
- 授权状态查询与刷新
- 试用期创建与过期判断
- 试用默认天数后台可配置
- 设备试用重置

### 5) 更新公告
用于向插件用户显示新版本提示。

功能点：
- 后台配置最新版本号与公告文案
- 配置“前往更新”跳转链接
- 插件侧对比本地版本并展示更新条

## 核心功能参数

### A. 归档参数（扩展侧）

- `mode`
  - 作用：归档模式选择
  - 可选值：`page`（当前页）、`site`（整站）

- `startUrl`
  - 作用：抓取起始地址
  - 说明：`site` 模式下为站点入口；`page` 模式下可指定页面

- `archiveName`
  - 作用：输出 ZIP 文件名
  - 说明：为空时自动生成

- `zipRootFolder`
  - 作用：ZIP 内部根目录名
  - 说明：用于归档文件组织

- `depth`
  - 作用：站点抓取深度
  - 范围：`0-20`

- `pageScope`
  - 作用：页面链接跟随范围
  - 可选值：
    - `same-origin`（同源）
    - `same-site`（同主站/子域）
    - `any-domain`（任意域名）

- `allowExternalResources`
  - 作用：是否允许下载外部域资源
  - 类型：`boolean`

- `includeScripts`
  - 作用：是否下载脚本资源
  - 类型：`boolean`

- `includeMedia`
  - 作用：是否下载媒体资源
  - 类型：`boolean`

- `render`
  - 作用：是否等待页面渲染后再抓取
  - 类型：`boolean`

- `renderTimeoutMs`
  - 作用：渲染等待时长（毫秒）
  - 范围：`0-45000`

- `sequential`
  - 作用：是否串行下载
  - 类型：`boolean`

- `maxConcurrency`
  - 作用：并发下载数量
  - 范围：`1-20`

- `quiet`
  - 作用：静默模式开关
  - 类型：`boolean`

- `proxyUrl`
  - 作用：代理地址输入参数位
  - 说明：当前版本预留字段，未启用真实代理转发

- `uiLanguage`
  - 作用：界面语言
  - 可选值：`auto`、`zh-CN`、`en-US`

### B. 授权参数（扩展/服务端）

- `activationCode`
  - 作用：激活码
  - 格式：`WD1-XXXX-XXXX-XXXX-XXXX`

- `licenseId`
  - 作用：授权记录唯一标识

- `deviceId`
  - 作用：设备唯一标识，用于试用/激活绑定

- `plan`
  - 作用：授权套餐标识（如 `pro`、`trial`）

- `maxActivations`
  - 作用：单个激活码可绑定设备上限

- `expiresAt`
  - 作用：授权到期时间（毫秒时间戳）

- `trialDaysDefault`
  - 作用：默认试用天数
  - 说明：仅对新创建的试用记录生效

### C. 更新公告参数（后台）

- `latestVersion`
  - 作用：最新版本号
  - 说明：用于与扩展本地版本对比

- `title`
  - 作用：公告标题

- `notes`
  - 作用：公告说明文本

- `url`
  - 作用：更新跳转地址（“前往更新”）

- `publishedAt`
  - 作用：公告发布时间（毫秒时间戳）

## 管理端能力参数

### 激活码生成参数
- `plan`：套餐标识
- `maxActivations`：设备上限
- `expiresAt`：到期时间（支持时间戳/相对天数/日期格式）

### 试用重置参数
- `deviceId`：目标设备 ID

### 列表展示关键字段
- `licenseId`：授权 ID
- `key`（activation code）：激活码明文（新生成记录可见）
- `status`：授权状态
- `plan`：套餐
- `max_activations`：上限
- `expires_at`：到期时间
- `created_at`：创建时间
