# WebDown 使用说明（中文）

## 工具作用

WebDown 用于把网页保存为可离线查看的本地包，支持：
- 保存当前页面
- 抓取整站并打包
- 导出 ZIP，便于本地备份、分享和归档

## 主要功能

- 双模式归档：当前页 / 整站抓取
- 资源完整下载：页面依赖的 CSS、JS、图片、字体、媒体
- 抓取范围控制：深度、域名范围、外部资源开关
- 抓取性能控制：串行/并发、并发数量、渲染等待
- 任务控制：开始、暂停、继续
- 状态可视化：当前阶段、当前 URL、下载统计
- 更新提示：有新版本时显示“前往更新”
- 中英文界面切换

## 怎么使用

### 1. 进入扩展面板
点击浏览器工具栏中的 WebDown 图标，打开操作面板。

### 2. 选择模式
- 保存当前页：只归档当前页面
- 整站抓取：从起始 URL 递归抓取

### 3. 配置抓取参数
常用参数说明：
- `起始 URL`：抓取入口地址
- `抓取深度`：整站模式下链接递归层级
- `页面范围`：同源 / 同站 / 任意域名
- `允许下载外部资源`：是否抓取跨域静态资源
- `下载第三方脚本`：是否包含脚本文件
- `下载音视频资源`：是否包含媒体文件
- `等待渲染后抓取`：动态页面建议开启
- `渲染等待(ms)`：渲染等待时长
- `串行模式`：按顺序下载，稳定但较慢
- `并发下载数`：并发数量，越大越快但压力越高
- `ZIP 文件名`：导出文件名（可自定义）
- `ZIP 根目录`：压缩包内根目录名称（可选）

### 4. 开始抓取
点击“开始”。抓取过程中可查看实时状态和统计数据。

### 5. 暂停与继续
- 点击“暂停”可中断后续抓取流程
- 点击“继续”从已有进度恢复，不会丢失已下载内容

### 6. 下载结果
任务完成后会自动生成并下载 ZIP 文件，解压后可离线浏览。

## 使用建议

- 静态站点：可提高并发，提升速度
- 动态站点：开启“等待渲染后抓取”并适当增加等待时间
- 大站点：先小深度测试，再逐步加深
- 网络波动时：可先暂停，稍后继续
